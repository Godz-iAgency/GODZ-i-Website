# Project Constitution — GODZ-i

## Behavioral Rules
1. **Tone**: Direct, confident, zero fluff. No passive or hedging language.
2. **Audience**: Marketing agency founders and operators ONLY.
3. **One CTA**: "Book a Free Strategy Session" — links only to https://cal.com/christopher-downer-6pkxir/strategy-session. No other CTAs.
4. **No Pricing**: Never mention pricing, tiers, or packages anywhere.
5. **No generic AI stock imagery**: All visuals must be custom, branded, or CSS-based.
6. **LinkedIn bidirectional**: The site links to LinkedIn; LinkedIn profile links back to site.
7. **No passive language**: Avoid "we might", "could help", "may be", etc.

## Architectural Invariants
- Single HTML file (index.html) + Single CSS file (styles.css)
- Font: Montserrat (via Google Fonts CDN)
- Colors: Primary #E8430A, Secondary #2B3A52, Background #FFFFFF, Light Grey #F5F5F5
- All 9 sections must be present and in order
- All CTA buttons trigger the same cal.com URL
- Mobile breakpoint at 768px
- No JavaScript frameworks; vanilla JS only if needed
