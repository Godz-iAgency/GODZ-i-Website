# Findings

## Research & Discoveries

### Project Type
Static single-page landing page — no backend, no APIs. Phase 2 (Link) is N/A.

### Technology Decisions
- Pure HTML + CSS — fastest possible load time, zero framework overhead
- Google Fonts CDN for Montserrat
- No JavaScript frameworks required
- No external API integrations needed

### Conversion Best Practices Applied
- Single CTA throughout (reduces decision fatigue)
- Problem → Solution → Proof → CTA structure (classic direct-response flow)
- Founder photo / personal story in About section builds trust
- Dark navy sections break visual monotony and create emphasis
- Orange (#E8430A) is high-contrast attention color for CTAs

### Constraints
- No pricing on page
- No generic AI imagery
- No passive language
- LinkedIn must be bidirectionally linked
- All CTAs → same cal.com link
